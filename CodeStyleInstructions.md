ИНСТРУКЦИЯ ДЛЯ ПЕРЕПИСЫВАНИЯ КОДА
Перепиши код в максимально простом и интуитивно понятном стиле, не меняя поведение и логику вообще, НЕ МЕНЯЯ ФУНКЦИОНАЛА И ЛОГИКИ.
Требования:
Никаких сложных конструкций
избегать тернарных операторов
избегать логических || в JSX
избегать неочевидных сокращений
Вся логика — в именованных функциях
навигация должна быть вынесена в function openSomething()
обработчики событий не писать прямо в JSX
Явность важнее краткости
лучше 10 строк, чем 1 “умная”
props передавать явно (title={event.title}), не через {...event}
Код должен быть понятен новичку
использовать обычные function, а не стрелки без необходимости
добавлять комментарии-блоки (// --------------------)
Структура файла
импорты
данные
функции-обработчики
подготовка JSX
return
styles
TypeScript не усложнять
использовать as never только там, где без этого не работает
не добавлять сложные типы
Цель: код должен читаться как пошаговая инструкция, а не как “React-магия”.
